# eMathica — AI Knowledge Base

`AI/` 是 eMathica 项目的长期知识库，不是临时报告堆放区。
它用于描述“项目是什么、为什么这样设计、下一步应该往哪里走”。

## 当前结构

- `Core/`：架构真相、对象系统、路线图
- `Data/`：机器可读的能力数据
- `ProductDesign/`：产品定义与用户视角说明

## 使用规则

- 当前真相放在 `Core/`、`Data/`、`ProductDesign/`
- 不把完成型 audit / report / plan 当成长期真相
- 不再把 `archive/` 或临时研究材料当成当前来源
- 如果需要临时分析，应控制在短生命周期文档里，而不是污染长期知识库

## 阅读顺序

1. `Core/Architecture.md`
2. `Core/ObjectSystem.md`
3. `Core/Roadmap.md`
4. `ProductDesign/README.md`
5. `Data/CapabilityRegistry.json`
